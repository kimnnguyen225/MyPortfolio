How to install & configure SQL Server 2019 Developer with 4.6.2 .NET framework on Windows 10



Disclaimer

Audience


This guide is intended to provide a fully functional database/SQL programming environment knowledge for beginner to intermediate level users.

Install SSMS & Database AdventureWorks2019 from Microsoft


Storage need: 6-8GB
Total installation time < 60 mins.
- You can sign up for Microsoft account if you want, I didn't sign-up for one in this guide.
- New SQL Server 2019 standalone install ~ 20 mins.
- SQL Server Management Tools ~ 15 mins.
- AdventureWorks sample online transaction Database (install and restore) ~ 10 mins.

Let's begin...

  1. Install SQL Server 2019: https://www.microsoft.com/en-us/sql-server/sql-server-2019#Install
  2. Install .NET framework 4.6.2: https://www.microsoft.com/en-us/download/details.aspx?id=53344
  3. Run SQL2019CTP2.5-SSEI-Eval.exe to install SQL Server 2019 to your Windows
  4. Click on New SQL Server stand-alone installation or add features to an existing installation


  5. Click Next


  6. Click on the 3 checked-boxes as shown in figure below. Then click Next


  7. Click Next


  8. Click Next


  9. Click on Add Current User. Then click Next


  10. Click Close


  11. Go back to SQL Server Installation Center. Click on Install SQL Server Management Tools


  12. Windows will open up a page, scroll down and find the link as shown in the figure below, click on it to download SQL Server Management Studio version 18.0


  13. Double click on SSMS-Setup to run it


  14. Click Install


  15. Create a folder and name it temp in your drive, in my example, I created the folder in my C:/Program Files/temp
    Go to: https://docs.microsoft.com/en-us/sql/samples/adventureworks-install-configure?view=sql-server-2017
    and look for AdventureWorks2017.bak, download it to your temp folder


  16. On your Windows, search for SQL Server Management Studio 18


  17. Once the application runs, it will prompt you to connect to server.
    For Server name, type localhost, then click Connect


  18. Right-click on Databases, choose Restore Database


  19. Click on the three ... ellipsis, then click on Add as shown in the figure below


  20. Look for temp folder, then click on the .bak file. Then click OK


  21. Click on OK


  22. Wait for it to restore the database, when it's finished, it will prompt you with a successfully restored message. Click OK.


  23. Follow step 18-22 if you want to restore your sample database if you messed it up while learning.
    Click on New Query, and run a simple select query to test the database to make sure it works perfectly.
